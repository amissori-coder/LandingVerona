# Loghi dei Partner

Salvare in questa cartella i seguenti file immagine (PNG con sfondo trasparente consigliato):

| File atteso             | Contenuto                                                       |
|-------------------------|-----------------------------------------------------------------|
| `logo-revilaw.png`      | Logo Revilaw completo (R + wordmark in blu navy)               |
| `logo-revilaw-mark.png` | Marchio "R" Revilaw in bianco, per la navbar scura             |
| `logo-advant-nctm.png`  | Logo Advant Nctm (ADVANT in rosso + Nctm in grigio)            |
| `logo-unindustria.png`  | Logo Unindustria "associato" con aquila                        |
| `logo-sole24ore.png`    | Logo Gruppo Il Sole 24 Ore (media partner)                     |

## Formati e dimensioni consigliate

- **Formato**: PNG con sfondo trasparente (oppure SVG)
- **Risoluzione**: almeno 2x per retina display
  - `logo-revilaw.png`: 400 × 120 px
  - `logo-advant-nctm.png`: 600 × 180 px
  - `logo-unindustria.png`: 600 × 240 px
  - `logo-revilaw-mark.png`: 128 × 128 px

## Come salvare i file

Copiare i file nelle rispettive posizioni:

```
29.04/
└── assets/
    ├── logo-revilaw.png
    ├── logo-revilaw-mark.png
    ├── logo-advant-nctm.png
    └── logo-unindustria.png
```

Una volta salvati, la landing page li mostrerà automaticamente al posto dei placeholder.
